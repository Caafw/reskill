# CAAFW Course Builder Skill

Generates a structured course outline for any subject using the CAAFW Course Builder.
Output is a ready-to-use interactive HTML file with modules, lessons, learning outcomes,
and assessments — branded for the Centre for Applied AI and Future of Work.

---

## What This Skill Does

Delivers a single-file interactive HTML course builder that walks the user through
five steps:

1. **Subject** — any academic topic, professional subject, or internal process domain
2. **Role** — Learner (self-study), Instructor (trainer delivery), or Instructional Designer (process documents to training)
3. **Level** — Beginner, Intermediate, or Advanced
4. **Format** — Full text, structured bullet points, or presenter talking points
5. **Generate** — AI produces a complete course outline with modules, lesson titles, learning outcomes, duration estimates, and module assessments

The free version (this file) generates the outline. Full lesson content, worked examples,
assessments, and complete course generation are delivered through CAAFW Re/Skill.

---

## Three User Roles

### Learner
Content written directly to the student. Explanations, self-check questions,
worked examples, and key term definitions.

### Instructor
Content written as a trainer delivery guide. Subject matter to teach,
teaching examples, discussion questions, misconceptions to address, timing guidance.

### Instructional Designer
For organisations converting internal process documents, SOPs, compliance guidelines,
or regulatory frameworks into structured staff training. Process chips trigger a
direct enquiry to CAAFW — this use case is built to the client's own documents,
not generic AI content.

**Enquiries for Instructional Designer / document-to-training:**
Contact hello@caafw.org or visit caafw.org/Reskill

## How to Use This Skill

When triggered, deliver the `caafw-course-builder-FREE.html` file from this repository
as an artifact or direct download. The file is self-contained — no server, no install,
no API key required when used inside Claude.ai.

For standalone use outside Claude.ai, the user will need an Anthropic API key.
The tool detects this and prompts accordingly.
